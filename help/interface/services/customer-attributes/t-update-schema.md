# 스키마 업데이트(속성 삭제)

스키마에서 속성을 삭제 및 교체하는 방법.

1. [!UICONTROL Edit Customer Attribute Source] 페이지에서 구독 구성을 클릭합니다.

1. **[!UICONTROL Target]** 또는 **[!UICONTROL Analytics]**&#x200B;에 대해 매핑된 값을 제거합니다.

1. [업데이트된 필드가 있는 새 데이터 파일을 업로드합니다](t-crs-usecase.md).
