# Creative Cloud 데스크탑에서 공유 폴더로 자산 업로드

1. 공유를 활성화합니다.
1. [!DNL Creative Cloud] 데스크톱을 실행하고 [!UICONTROL 자산으로 이동합니다].
1. 폴더 **[!UICONTROL 열기를]** 클릭합니다.
1. 운영 체제의 파일 검색을 사용하여 자산을 선택하거나 이동합니다.
